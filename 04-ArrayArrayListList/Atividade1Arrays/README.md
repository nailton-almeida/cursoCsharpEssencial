# Array, Arraylist, List
##  Atividade sobre Array

Nessa atividade foi exercitado a utilização de array para o armazenamento de informações. Abaixo estão as orientações do exercício.

- Crie um programa ```C#``` que recebe via teclado o nome de cinco alunos e armazene os nomes em um array chamado nomes, e, a seguir receba também via teclado a nota, como um double, dos cincos alunos e armazene os valores em um array chamado notas.

- A seguir realize as seguintes operações:		
       
     1. Acessar e exibir no console os nomes dos alunos armazenados no array nomes; 
     2. Acessar e exibir no console as notas dos alunos armazenados no array notas; 
     3. Calcular a média aritmética das notas e exibir no console o valor obtido.
