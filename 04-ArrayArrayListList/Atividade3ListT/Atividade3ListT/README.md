# Array, Arraylist, List
##  Atividade sobre List\<T>

Nessa atividade foi exercitado a utilização de array\<T> para o armazenamento de informações. Abaixo estão as orientações do exercício.

Dada uma classe Aluno definida com as propriedades Nome(Tipo String) e Nota(Tipo Double), criei uma lista de objetos do tipo Aluno usando a coleção \<T> com os seguintes dados que representam o nome e a nota do aluno:

| Nome   | Nota       |
| :---------- | :--------- |
|Maria  |  8.75|
|Manoel | 6.98|
|Amanda | 7.25|
|Carlos | 6.55|
|Jaime  |8.50|
|Debora | 5.98|
|Alicia | 9.25|
|Sandra | 5.55|
|Marta  | 7.85|
|Sueli  | 9.15|
