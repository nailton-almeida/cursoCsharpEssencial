# Array, Arraylist, List
##  Atividade sobre Array Bidimensional

Nessa atividade foi exercitado a utilização de array bidimensional para o armazenamento de informações. Abaixo estão as orientações do exercício.

- Declare e aloque memória para um array de strings chamado alunos com 2 linhas e 5 colunas, a seguir realize as seguintes operações:

	1 - Inicializar o array com dados referente aos nomes dos alunos recebidos via teclado informando o índice da linha e coluna de cada elemento que está sendo recebido;

	2 - Exibir no console os dados informados no array bem como o índice da linha e coluna de cada elemento;

	3 - Usar os nomes dos alunos a seguir respeitando a ordem indicada:

	1. Maria, Paulo, Marta, Pedro, Carlos
	2. Silvia, Amanda, Manoel, Paula, Alicia
