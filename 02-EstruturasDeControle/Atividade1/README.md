# Estruturas de Controle do C#

## Atividade 1

 Nessa atividade foram abordados as estruturas de controle disponíveis na ```linguagem C#```, como o if, switch, while e for. Abaixo estão as questões propostas que possuem os mesmo nomes das resoluções que se encontram no diretório ```Exec```.

- Exec001 - Escreva um programa para receber 3 números inteiros e a seguir calcular e exibir qual deles é o maior;

- Exec002 - Escreva um programa para calcular a raiz da equação quadrática;

- Exec003 - Escreva um programa para exibir os 10 primeiros números naturais e calcular a sua soma usando os loop while, do-while e for;

- Exec004 - Escreva um programa para exibir a tabela de multiplicação de um número natural maior que zero recebido via teclado. Siga as premissas abaixo:
    
      1. Verifique se o número é maior que zero e emita uma mensagem;
        
      2. Considere a tabela de multiplicação de 1 até 10;

      3. Após exibir a tabela torne a solicitar outro número;

      4. Para sair do loop defina uma condição de saída.

- Exec005 - Escreva um programa para solicitar ao usuário que escolha a resposta correta de uma lista de opções de resposta de uma pergunta. O usuário pode optar por continuar respondendo a pergunta ou parar de responder;

- Exec006 - Crie um programa para exibir no console os números pares de 10 a 20, ambos incluídos, exceto 16, de 3 maneiras diferentes. Siga as premissas abaixo:

        1. Incrementando 2 em cada passo;
        2. Incrementando 1 em cada passo;
        3. Com e loop infinito (use "break" e "continue").

- Exec007 - Escreva um programa para exibir o padrão como triângulo de ângulo reto usando asterisco (*);

- Exec008 - Escreva um programa para calcular o fatorial de um número inteiro;

- Exec009 - Escreva um programa para exibir as tabelas de multiplicação do 2 ao 6 usando o loop do-while aninhado;

- Exec0010 - Crie um programa que recebe na entrada de dados um número inteiro de 0 a 10 que representa a nota de um aluno. Com base na tabela a seguir imprima no console qual o resultado da avaliação do aluno. (Use a instrução switch, break e default em um loop infinito e defina uma condição de saida.)

- Exec0011 - Crie um programa para realizar as operações de adição, subtração, multiplicação e divisão de números inteiros. (considere que na divisão podemos ter números fracionários e que não existe divisão por zero e quando isso ocorrer exibir uma mensagem de alerta).
